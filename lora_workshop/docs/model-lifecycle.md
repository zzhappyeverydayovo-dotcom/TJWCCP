# 模型全生命周期纳管说明

## 纳管对象

- 核心图像生成大模型：天津地域文化文创专属底座模型。
- 定制 LoRA 矩阵：不少于 50 款可插拔 LoRA。
- 扩展模型能力：视频生成、3D 建模、沉浸式交互和产品打印对接相关模型。

## 模型台账字段

每个模型或 LoRA 至少记录：

- 模型编号、名称、版本、类型、适用场景。
- 底座模型、训练数据版本、授权证明状态。
- 触发词、默认权重、推荐参数、禁用参数。
- 训练配置、训练日志、checkpoint 校验结果。
- 效果评估报告、人工审核结论、发布状态。
- 调用权限、调用次数、响应时延、异常记录。

## 版本状态

```text
draft -> internal_preview -> review_pending -> approved_for_platform -> retired
```

`approved_for_platform` 必须满足：

- 数据授权证明完整。
- checkpoint 可加载。
- 样张通过风格还原、文化准确性、产品适配性评估。
- 无高风险版权、肖像、商标和敏感内容问题。
- 已生成使用说明、限制范围和回滚记录。

## 与现有平台模块的关系

- 非遗文创创作智能体：根据文化主题、产品品类、人群、场景、风格、色彩选择 LoRA 组合。
- 提示词优化智能体：输出正负向提示词、采样参数和 LoRA 权重。
- 审核辅助智能体：对生成图像做文化表达、版权肖像、商标和公开展示风险校验。
- 运营分析智能体：统计模型调用量、生成成功率、热门文化元素、审核通过率。
- 系统管理：通过 RBAC 控制模型调用、组合叠加、训练任务和发布权限。
